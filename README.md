# Pine Paw Retreat – Pet Boarding Management System

Pine Paw Retreat is a web application designed to simulate the operational needs of a pet boarding facility, including managing pet owners, animals, and boarding reservations. This project is meant to cover both internal systems for the pet boarding facility and give consumers a portal to manage their upcoming appointments.

This project is still underdevelopment.

## Tech Stack

Backend: ASP.NET MVC (.NET Framework)

Language: C#

Database: SQL Server, Entity Framework

Frontend: Razor Views, HTML, CSS, Bootstrap
